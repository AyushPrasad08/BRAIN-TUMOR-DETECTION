Frontend (React + TypeScript)

Setup

1. cd frontend
2. npm install
3. npm run dev

Features
- React + TypeScript
- Tailwind CSS
- Framer Motion
- React Query
- Left sidebar layout
- Pages: Dashboard, Upload, History, About

Notes
- API calls point to /api and are mocked in services/mockData.ts. When backend available, configure proxy or set VITE_API_URL.
